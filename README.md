# 🎵 SmartRandomPlayer

レベルベースの重み付きランダム選曲ができるWebアプリケーション

## 📱 アプリケーションについて

お気に入りの曲が多めに、聞き飽きた曲は少なめに。  
「完全ランダムじゃ物足りない」を解決する音楽プレーヤーです。

### 主な機能
- 🎲 レベル1〜5の重み付きランダム選曲
- 📊 再生回数による自動レベル調整
- 💾 ブラウザ内でのデータ永続化
- 🎯 選曲確率のカスタマイズ

### 対応ブラウザ
- Google Chrome（推奨）
- Microsoft Edge
- Opera
